使用 指南

从开箱到上手，三步搞定。以下是详细的安装和使用教程。

快速 开始

无论你是个人用户还是企业用户，都能在几分钟内完成部署

1

插入U盘

将UnitClaw便携AI盘插入电脑的USB接口（支持USB-A和USB-C双口）。

支持的操作系统：Windows 10/11、macOS 12+、Ubuntu 20.04+、Debian 11+

2

双击启动

打开U盘根目录，根据你的操作系统双击对应的启动脚本：

Windows

双击 UnitClaw.bat

macOS

双击 UnitClaw.command

Linux

终端运行 ./UnitClaw.sh

3

选择安装

启动菜单会自动弹出，选择"一键安装"即可。安装过程中无需任何操作，等待完成即可。

首次安装约需 3-5 分钟（取决于网络和电脑性能）

启动菜单 功能

UnitClaw启动菜单提供完整的生命周期管理

一键安装

自动安装OpenClaw框架和所有预装AI执行单元

启动UnitClaw

快速启动已安装的AI执行单元平台

备份还原

一键备份你的配置和数据，支持快速还原

修复工具

自动检测和修复常见问题，恢复到正常状态

AI执行单元 使用

安装完成后，你可以自由选择和组合AI执行单元

添加执行单元

启动UnitClaw后，进入"单元市场"界面，浏览可用的AI执行单元列表，点击"安装"即可添加到你本地。所有安装操作均离线完成，无需联网。

组合工作流

在"工作流编辑器"中，你可以将多个AI执行单元拖拽连接，构建自动化工作流。例如：数据抓取单元 → 文案生成单元 → 自动发布单元，实现一键内容生产。

配置模型

在设置页面配置你偏好的AI模型。UnitClaw支持 DeepSeek、Kimi、通义千问、智谱GLM 等 8+ 国产模型，也支持 Claude、GPT 等国际模型（需API Key）。

连接平台

支持对接 QQ、飞书、微信、Telegram、Discord、钉钉 等聊天平台。在"平台接入"页面按照指引配置即可，通常只需 3 条命令即可完成。

企业版 部署指南

企业部署包支持本地服务器和私有云部署

环境准备

  • Linux服务器（推荐 Ubuntu 22.04）
  • 最低 8GB 内存、50GB 磁盘
  • 内网或局域网环境
  • Docker 20.10+

一键部署

  • 解压部署包到服务器
  • 运行 deploy.sh
  • 自动配置环境和服务
  • 约 10 分钟完成

配置管理

  • Web管理后台
  • 用户权限管理
  • 单元市场管理
  • 系统监控面板
